SendAutoWP

Established a http connection to the server sending the parameters via POST. It waits until the Server complete the Sending (using the serverGeneralError or the serverResultPost functions) and return a value depending of the Response of the Server.

public String SendAutoWP(String Login, String Pwd, String URL, String Text) ;

Parameters
Login
  String with sender's telephone number

Pwd
  String with sender's password

URL
  String with the URL where the POST petition is going to be sent

Text
  String with the text which is going to be sent to the URL.

Return
0 OK.
1 If login format not correct.
2 URL is not correct.
3 Password format is not correct.
4 User is not allowed to use this service.
5 User does not exist.
6 Password is not correct.
7 User is spired.
8 Text format is not correct.
9 Connection cannot be stablished
-2 Error: The connection with the Server cannot be stablished.
-3 Error: Non security error.
