Polling

Wait for a response of the SMS 2.0 platform.

public String Polling();

Parameters
-
Return
String with a XML structure with the followings posibles responses:
  • Empty string - no body in the response

  • An authorization request. One user wants to be added to your contact list. Use isNewContact(String XMLPolling) to check it, GetUserIDNewContact(String XMLPolling) to get the User ID (telephone number) and GetTransactionIDNewContact(String XMLPolling) to get the Transaction ID needed to authorise the contact.

  • A new message. One user has sent you a message. Use IsNewMessage(String XMLPolling) to check it, GetUserIDMessage(String XMLPolling) to get the user ID (telephone number) and GetMessage(String XMLPolling) to get the message.

  • A contact presence list. A status of the presence of the contacts has been changed. You will receive a new list with all the states of the contacts. Use IsPresenceList(String XMLPolling) to check it and GetPresenceList(String XMLPolling) to get a vector with the contacts.
